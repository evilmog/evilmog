# intro
this converts RTTTL to stuff usable in circuit python for foxes

# usage
```
python3 rtttl.py "Wannabe:d=4, o=5, b=125:16g, 16g, 16g, 16g, 8g, 8a, 8g, 8e, 8p, 16c, 16d, 16c, 8d, 8d, 8c, e, p, 8g, 8g, 8g, 8a, 8g, 8e, 8p, c6, 8c6, 8b, 8g, 8a, 16b, 16a, g"
```

this would convert "wannabe" into something usable in your project
